# Estudos de Rust 🦀

Este repositório é uma coleção de pastas usadas para organizar os códigos que estou escrevendo enquanto estudo a linguagem **Rust**.

## Sobre

Cada pasta corresponde a uma parte dos meus estudos, seguindo o conteúdo do livro oficial da linguagem, [*The Rust Programming Language*](https://doc.rust-lang.org/book/) (disponibilizado gratuitamente pela equipe do Rust). A ideia é manter os exemplos, exercícios e anotações organizados conforme avanço pelos capítulos.

## Organização

O repositório está estruturado como uma sequência de pastas, cada uma agrupando os códigos relacionados a um tópico ou capítulo do livro. Conforme o estudo progride, novas pastas são adicionadas.

## Objetivo

Servir como registro pessoal de aprendizado, facilitando a revisão dos conceitos e o acompanhamento da evolução ao longo do livro.
